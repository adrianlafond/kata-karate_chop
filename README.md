# kata karate chop

[Kata Karate Chop](http://codekata.com/kata/kata02-karate-chop/)

## Installation
Clone and then run:
```
npm install
```

## Commands

### `npm test`
Run the tests once.

### `npm start`
Run the tests in watch mode.

### `npm run lint`
Lint the code once.

### `npm run lint:watch`
Lint the code in watch mode.
